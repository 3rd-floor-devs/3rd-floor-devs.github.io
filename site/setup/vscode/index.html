    VS Code 기본 설정 및 사용법

    Visual Studio Code(VS Code)는 현대적인 개발을 위한 가장 강력하고 유연한 코드 에디터입니다. 원활한 파이썬 개발을 위한 핵심 설정법을 알아봅니다.

    1. 필수 확장 프로그램(Extension) 설치

    VS Code를 처음 설치하면 아주 기본적인 기능만 들어있습니다. 파이썬 파일을 정상적으로 불러오고 스마트한 기능을 사용하려면 다음 확장을 반드시 설치해야 합니다.

    1. VS Code 왼쪽 사이드바에서 Extensions 아이콘(사각형 아이콘)을 클릭합니다.
    2. Python을 검색하여 Microsoft에서 만든 Python 확장 프로그램을 설치합니다.
    3. 이 확장이 있어야 코드 자동 완성, 디버깅, 가상환경 연동 등이 가능해집니다.

    2. 프로젝트 시작하기: 폴더 열기 (Open Folder)

    VS Code에서 가장 중요한 습관은 '폴더 단위'로 작업하는 것입니다.

    • 방법: File > Open Folder... 메뉴를 클릭하고, 내가 작업할 프로젝트 폴더를 선택합니다.
    • 왜 폴더를 여나요?: 개별 파일만 열면 프로젝트 전체의 흐름을 파악하기 어렵고, 가상환경 설정이나 테스트 도구가 제대로 작동하지 않습니다.

    워크스페이스(Workspace) 이해하기

    폴더를 열면 VS Code는 해당 폴더를 하나의 워크스페이스(작업 공간)로 인식합니다. 이 공간 안에 설정값(.vscode 폴더 등)을 저장하여, 다음에 폴더를 열었을 때 내가 설정한 환경을 그대로 불러올 수 있게 됩니다.

    3. 터미널 활용하기 (Ctrl + Shift + `)

    코드를 작성하는 것만큼 중요한 것이 명령어를 입력하는 것입니다.

    • 단축키: Ctrl + Shift + ~ (또는 백틱 `)
    • 용도: 별도의 CMD나 PowerShell 창을 띄울 필요 없이, VS Code 하단에서 바로 명령어를 입력할 수 있습니다. uv 명령어나 파이썬 실행 명령을 여기서 입력하게 됩니다.

    4. 파이썬 인터프리터 설정 (가상환경 연결)

    프로젝트마다 생성한 독립적인 가상환경을 VS Code와 연결해 주는 매우 중요한 단계입니다.

    1. Ctrl + Shift + P를 눌러 커맨드 팔레트를 엽니다.
    2. Python: Select Interpreter를 검색하여 선택합니다.
    3. 목록에서 현재 프로젝트의 가상환경(.venv) 경로가 포함된 파이썬 버전을 선택합니다.

    ⚠️ 주의: 목록에 가상환경이 보이지 않는다면, 먼저 가상환경이 생성되어 있어야 합니다. uv 설치 및 가상환경 생성 가이드를 참고하여 가상환경을 먼저 만들어주세요.

    요약 📝

    1. Python Extension: 파이썬 개발의 심장이자 엔진
    2. Open Folder: 프로젝트 단위 작업의 시작점
    3. Terminal: 키보드만으로 개발을 가속화하는 창
    4. Select Interpreter: 내 프로젝트가 어떤 가상환경에서 실행될지 결정하는 단계
