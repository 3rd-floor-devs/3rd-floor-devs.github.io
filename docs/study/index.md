## 🏗️ 커리큘럼 로드맵

| 단계 | 주제 | 기간 | 핵심 목표 |
| :--- | :--- | :--- | :--- |
| **Phase 1** | 에이전트의 뇌와 눈을 만들다 | 1~3주 | LangGraph 기초 및 시각 지능(OCR/Vision) 구현 |
| **Phase 2** | 에이전트의 귀와 기억 | 4~5주 | 음성 인터페이스(STT/TTS) 및 RAG/메모리 구축 |

| **Phase 4** | 지휘관 에이전트 | 9~10주 | 멀티 에이전트 아키텍처 및 MCP 활용 |
| **Phase 5** | 파이널 프로젝트 | 11~12주 | 종합 사내 업무 자동화 비서 구축 |

### 💡 스터디 운영 팁

1. **지루함 방지:** 2주차(OCR)와 3주차(Vision)를 초반에 배치하여, 텍스트만 오가는 터미널 창을 벗어나 **"눈에 보이는 결과물"**을 빨리 만들게 하세요.
2. **데이터 준비:** SAP나 dataPARC는 실제 운영 서버 접근이 어렵다면, 엑셀이나 간단한 Mock DB(SQLite)로 구조만 똑같이 만들어서 실습하는 것이 좋습니다.
3. **환경:** Python 환경(LangChain, LangGraph)을 기본으로 하되, Azure 관련 실습은 크레딧(비용) 관리가 필요하므로 공용 계정이나 Free Tier 범위를 미리 확인하세요.
