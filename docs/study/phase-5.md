# Phase 5: 파이널 프로젝트 (종합 구축)

## 11주차 ~ 12주차: 사내 업무 자동화 비서 만들기

* **주제 예시:** **"스마트 설비 관리 비서"**
    1. 현장 작업자가 설비 사진을 찍어 업로드 (Vision/SAM)
    2. 음성으로 "이거 저번에 언제 고쳤지?" 물어봄 (Speech/STT)
    3. 에이전트가 이미지를 인식해 설비 번호 추출 (OCR or Vision)
    4. SAP/dataPARC에서 해당 설비 이력 및 현재 센서값 조회 (Tools)
    5. "지난달 5일에 베어링 교체했고, 현재 온도는 정상입니다." 라고 음성 및 텍스트로 답변 (TTS/GenAI)
    6. 작업자가 "점검 완료 처리해줘" 하면 관리자 승인 요청 후 SAP 입력 (HITL)
