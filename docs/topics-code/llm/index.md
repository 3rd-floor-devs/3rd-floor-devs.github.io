# LLM (Large Language Models) 🤖

인공지능의 핵심 기술인 **거대 언어 모델(LLM)**을 활용하여 지능형 애플리케이션을 구축하는 방법을 학습합니다. 가장 널리 쓰이는 프레임워크인 LangChain과 LangGraph를 중심으로 기초부터 실전까지 다룹니다.

![LLM 메인 이미지](./images/llm_hero.png)

---

## 🏗️ 학습 커리큘럼

LLM을 정복하기 위한 단계별 가이드를 제공합니다.

### 🟢 기초 과정 (Basic)
LLM 애플리케이션 개발의 근간이 되는 필수 개념과 도구들을 학습합니다.
- [**LangChain 기본개념**](./basic/langchain-basics.md): LLM 호출부터 체인 구성, 구조화된 출력까지의 기초.
- [**LangGraph 기본개념**](./basic/langgraph-basics.md): 상태 관리를 기반으로 한 복잡한 워크플로우 설계법.

### 🔴 심화 과정 (Advanced)
실전 프로젝트에서 맞닥뜨리는 복잡한 문제들을 해결하기 위한 고급 기법들을 다룹니다.
- *(현재 준비 중입니다. 조만간 흥미로운 주제들로 채워질 예정입니다!)*

---

## 🛠️ 주요 도구들
우리가 주로 사용하게 될 핵심 도구들입니다.

- **LangChain**: LLM 구성 요소를 연결하는 사슬(Chain) 프레임워크.
- **LangGraph**: 순환(Cycle)이 포함된 에이전트 기반 시스템 구축 도구.
- **Arize Phoenix**: 실행 과정을 투명하게 관찰하는 오토 트레이싱 도구.

시작할 준비가 되셨나요? 먼저 [**LangChain 기본개념**](./basic/langchain-basics.md)부터 차근차근 시작해 보세요! 🚀
